How to change letter spacing in Canva?
To change the letter spacing in Canva, first, you need to select the text box that contains the text you want to adjust. Once you've selected the text box, you'll see a toolbar at the top of the screen. Look for the 'Spacing' option in this toolbar. Click on it to reveal the options for letter spacing and line height.

To set line spacing in Canva, you first need to select the text box that contains the text you want to adjust. Once you've selected the text box, you'll see a toolbar at the top of the screen. Look for the line spacing icon, which looks like two horizontal lines with arrows pointing up and down.

Click on the line spacing icon to open the line spacing options. You can either choose from the preset options or manually adjust the line spacing by clicking on the plus or minus buttons. Once you're satisfied with the line spacing, click outside the text box to apply the changes. Remember, you can always adjust the line spacing again if needed. Click and drag this slider to the left to decrease the letter spacing, or to the right to increase it. The changes will be applied immediately, so you can see the effect on your text as you adjust the slider.
